# Movie Search App

A React application for searching and browsing movies using an external API.

## Features

- Search movies by keyword
- View trending movies
- Detailed movie information
- Navigation with React Router
- Lazy loading and code splitting

## Tech Stack

- React
- React Router
- Axios
- CSS Modules
